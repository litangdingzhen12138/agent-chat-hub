## v1.0.0 (2026-06-25)

### 🎉 首次发布

#### ✨ 核心功能
- **自动检测** - 自动扫描本地 Claude Code、Codex CLI、OpenCode 数据
- **目录树** - 按项目目录层级组织会话，支持多级嵌套
- **聊天气泡** - 用户问题靠右，AI 回答靠左
- **Markdown 渲染** - 完整支持代码语法高亮
- **工具调用折叠** - 思考过程、工具调用默认折叠，点击展开
- **全文搜索** - 跨会话搜索消息内容
- **文件/文件夹上传** - 支持上传单个文件或整个文件夹

#### 🔍 支持的 Agent 工具
- 🟠 **Claude Code** - 自动检测 `~/.claude/projects/`
- 🔵 **Codex CLI** - 自动检测 `~/.codex/sessions/` 和 `history.jsonl`
- 🟢 **OpenCode 及衍生版** - 自动扫描 `~/.local/share/` 下所有子目录

#### 🎨 界面特性
- 深色主题，护眼舒适
- 可拖动调整侧边栏宽度
- 一键折叠/展开所有目录
- 每个 Agent 独立颜色标识
- 会话标题自动取用户第一段问题

#### 🛠️ 技术实现
- 后端: Python + FastAPI
- 前端: HTML + Tailwind CSS + Marked.js
- 无需 Node.js，纯 Python 运行
- 自动端口查找，避免冲突
- 跨平台支持 (Windows / macOS / Linux)

## 版本说明

### 版本号规则
采用 [语义化版本](https://semver.org/lang/zh-CN/)：
- **主版本号** - 不兼容的 API 修改
- **次版本号** - 向下兼容的功能性新增
- **修订号** - 向下兼容的问题修正

### 更新日志格式
参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)：
- `✨ 新功能` - 新增功能
- `🐛 修复` - Bug 修复
- `📝 文档` - 文档更新
- `🎨 界面` - UI/UX 改进
- `⚡ 性能` - 性能优化
- `🔧 其他` - 其他改动
